### 삽입 정렬

>각 숫자를 적절한 위치에 삽입하는 것

<br>

`*1* 10 5 8 7 6 4 3 2 9`

- 첫 번째 위치는 애초에 정렬이 되어있다고 생각

<br>

`_1_*10* 5 8 7 6 4 3 2 9`

- 1보다 크기 때문에 위치 유지하고, 다음 정렬할 원소로 이동.

<br>

`_1_10_*5* 8 7 6 4 3 2 9`

`1 *5* 10 8 7 6 4 3 2 9`

- 앞은 이미 정렬이 되어있고, 5의 자리만 찾아서 삽입하면 됨.

- 앞의 값들과 비교해서 5보다 큰 값이 왼쪽에 있다면 위치를 서로 바꿔줌.

- 5보다 큰 10이 왼쪽에 위치하기 때문에 위치를 바꿔줌.

- 1은 5보다 작기때문에 다음 정렬할 원소로 이동.

<br>

...

<br>

`_1_5_7_8_10_*6* 4 3 2 9`

`1 5 *6* 7 8 10 4 3 2 9`

- 앞이 이미 정렬이 되어있고, 6의 자리만 찾아서 삽입하면 됨.

- 앞의 값들과 비교해서 6보다 큰 값이 왼쪽에 있다면 위치를 서로 바꿔줌.

- 6보다 큰 10이 왼쪽에 위치하기 때문에 위치를 바꿔줌.

- 6보다 큰 8이 왼쪽에 위치하기 때문에 위치를 바꿔줌.

- 6보다 큰 7이 왼쪽에 위치하기 때문에 위치를 바꿔줌.

- 5는 6보다 작기때문에 다음 정렬할 원소로 이동.

- 이때 1은 비교할 필요없음 => 필요한 만큼만 비교함

<br>

...

<br>

**정렬 과정**

1. 현재 타겟이 되는 숫자와 이전 위치에 있는 원소들을 비교함.
(첫 번째 타겟은 두 번째 원소부터 시작함. == 첫 원소는 이미 정렬되어 있다고 생각)

2. 타겟이 되는 숫자가 이전 위치에 있던 원소보다 작다면 위치를 서로 교환함.
=> 타겟이 되는 숫자보다 큰 숫자가 왼쪽에 위치한다면 위치를 교환함.

3. 그 다음 타겟을 찾아 위와 같은 방법으로 반복.

<br>

**코드**

```
package sort;

import java.util.Arrays;

public class InsertionSort {
    public static void main(String []args){
        int temp, j;
        int [] array = {1, 10, 5, 8, 7, 6, 4, 3, 2, 9};

        System.out.println("정렬 전");
        System.out.println(Arrays.toString(array));

        for(int i = 0; i<array.length-1; i++){
            j = i; // 현재 정렬할 원소 선택
            while(array[j] > array[j+1]){ // 왼오를 비교, 왼쪽이 더 크다면 반복
                temp = array[j];
                array[j] = array[j+1];
                array[j+1] = temp;
                j--; // 1씩 빼가면서 앞 원소와 비교
            }
        }

        System.out.println("정렬 후");
        System.out.println(Arrays.toString(array));
    }
}
```
